# MELI Backend

## Librerías implementadas

- NodeJS
- Express generator
- Axios
- CORS

## Cómo iniciar la aplicación

### `cd meli-backend`
### `npm start`

**Monta la aplicación en el puerto [http://localhost:5000](http://localhost:5000)**

## Endpoints disponibles

#### Search

`/api/items?search=string`

#### Detalle de producto

`/api/items/:id`
